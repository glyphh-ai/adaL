# Legacy artifacts (2026-05 paper era) — kept as the audit record

This directory holds the **data artifacts** from the deleted 2026-05
white paper: Wikipedia extraction outputs for chemistry/philosophy/law
(~41K entries with `composed_of` decompositions), the consolidated
lattice files, and the cross-domain reports.

They are retained for provenance: the internal audit that led to this
project's rewrite (HDC removal, pre-registered re-benchmarking) was
conducted against these artifacts, and the audit's claims should remain
checkable. **Nothing in the current runtime, paper, or benchmarks
depends on this directory.** The paper-era benchmark and teach scripts
were removed (they targeted deleted modules and a discredited
methodology).

The current, maintained benchmark suite is `benchmark/phase{1,2,3}/`
with `benchmark/PROTOCOL.md`.
